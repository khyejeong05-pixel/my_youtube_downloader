# 🎥 Antigravity 유튜브 다운로더 (YouTube Downloader)

GitHub의 [Hitomi-Downloader](https://github.com/KurtBestor/Hitomi-Downloader)의 직관적이고 완성도 높은 다운로드 큐 관리 및 UI를 모티브로 하여 개발한 **한국어 버전 유튜브 다운로더 프로그램**입니다.

이 프로그램은 Python의 `customtkinter`를 사용하여 현대적이고 세련된 다크 테마 GUI를 제공하며, 백그라운드 멀티스레딩 기술과 `yt-dlp` 엔진을 탑재하여 GUI가 멈추지 않고 신속하게 동영상을 다운로드합니다.

---

## 🌟 주요 기능 (Key Features)

*   **다국어/한국어 최적화**: 모든 UI 레이블 및 상태 메시지가 자연스러운 한국어로 제공됩니다.
*   **포맷 및 화질 선택**: 최고 화질 비디오(MP4), 1080p, 720p, 480p 해상도 지정 및 고음질 MP3/M4A 음원 추출 기능을 지원합니다.
*   **재생목록(Playlist) 일괄 추가**: 유튜브 재생목록 링크를 붙여넣으면 리스트의 각 영상을 개별 작업으로 자동 분할하여 큐에 추가합니다.
*   **멀티스레드 다운로드 & 큐 관리**:
    *   동시 다운로드 개수를 설정하여 대기 작업이 자동으로 순차 실행되는 큐 관리 시스템입니다.
    *   개별 작업별 썸네일, 제목, 상태바, 다운로드 속도 및 실시간 남은 시간(ETA)을 시각적으로 모니터링합니다.
    *   동작 중인 개별 다운로드를 안전하게 취소하고 임시 파일을 정리하거나, 완료된 작업을 리스트에서 삭제할 수 있습니다.
*   **클립보드 감시**: 클립보드에 유튜브 주소 복사 시 자동으로 프로그램 입력칸에 적용하는 편의 기능이 내장되어 있습니다.
*   **엔진 자동 업데이트**: YouTube의 사이트 변경으로 다운로드가 막힐 경우를 대비해, 원클릭으로 내부 `yt-dlp` 엔진을 최신 버전으로 업그레이드할 수 있는 버튼을 제공합니다.

## 🚀 시작하기 (How to Run)

### 1. 사전 요구사항 (Requirements)
*   Python 3.10 이상이 컴퓨터에 설치되어 있어야 합니다.
*   필수 파이썬 라이브러리 설치가 필요합니다.

### 2. 패키지 설치
터미널(또는 CMD/PowerShell)을 열어 아래 명령어를 실행합니다.
```bash
pip install -r requirements.txt
```

### 3. 프로그램 실행
```bash
python main.py
```

## 🛠️ 고화질 및 오디오 인코딩용 FFMPEG 안내
유튜브의 1080p 이상 고화질 비디오는 비디오 스트림과 오디오 스트림이 나누어 제공되기 때문에 다운로드 후 병합 과정이 필요합니다. 또한 비디오에서 MP3 음원을 안정적으로 추출하기 위해서도 **ffmpeg** 프로그램이 필요합니다.

*   `ffmpeg`가 시스템에 설치되어 있지 않은 경우, 프로그램은 **720p 이하의 비디오(이미 비디오와 오디오가 인코딩된 상태의 최상위 화질)**로 안전하게 다운로드하도록 자동 대체(Fallback) 작동합니다.
*   **1080p 고화질 및 MP3 변환**을 원하신다면 아래 방법에 따라 `ffmpeg`를 설치하시는 것을 추천합니다.

### Windows에서 ffmpeg 설치하는 방법
1.  [ffmpeg 공식 웹사이트](https://ffmpeg.org/download.html) 또는 [BogoToBogo FFmpeg 빌드](https://github.com/BtbN/FFmpeg-Builds/releases)에서 Windows용 static 빌드 zip 파일을 다운로드합니다.
2.  압축을 푼 뒤 `bin` 폴더 경로(예: `C:\ffmpeg\bin`)를 시스템 **환경 변수(PATH)**에 추가합니다.
3.  터미널에서 `ffmpeg -version`을 실행하여 올바르게 인식되는지 확인합니다.
4.  유튜브 다운로더 프로그램을 재실행하면 경고 문구가 사라지고 고화질 결합 및 오디오 추출 기능이 정상적으로 활성화됩니다.
